Albino คืออะไร

Albino คือ Google Chrome extension ใช้อ่านกระทู้และความเห็นในเว็บ Pantip

แล้วทำไมเข้า Pantip ปกติไม่ได้

ได้ แต่ Albino เร็วกว่าเพราะ Albino จะโหลดเฉพาะข้อมูลที่จำเป็นต่อการอ่าน มาแสดงผลให้อ่านง่าย อ่านนานได้ ไม่ปวดตา

อยากใช้ Albino ต้องทำไง

โหลด Albino แล้วทำตามวิธีนี้ เดี๋ยวในอนาคตจะทำให้โหลดผ่าน Chrome Web Store ง่าย ๆ แต่ตอนนี้ยังเป็น beta อยู่ รออีกนิดดด

ทำไมต้องทำเป็น Chrome extension

คือเว็บปกติเนี่ยจะไม่มีสิทธิ์เข้าถึงเนื้อหาของเว็บอื่น เพราะเหตุผลด้านความปลอดภัย (ศัพท์เทคนิกคือ Same-origin policy) แต่ extension จะสามารถทำได้ โดย extension นี้สามารถเข้าถึงเนื้อหาในเว็บ Pantip (และเฉพาะเว็บ Pantip เท่านั้น)

งี้ก็ไม่ปลอดภัยสิ ถ้าเอาข้อมูลไปทำมิดีมิร้ายจะรู้ได้ไง

รู้ได้ เพราะ Albino เป็น open source หมายถึงใครก็สามารถเข้าไปส่องโค้ดได้ตามสบาย รับรองข้อมูลปลอดภัยหายห่วง
